Monte Carlo to Perform Volume Integral Over Earth; Save Interaction Locations for Flavor Dependent Weights Later¶

Integration Using Arrays¶

Perform Preferential Sampling to save time on finding an accurate value¶

The goal of a Monte Carlo integration is to perform the integral $I = \int f( x) d x$ where $x$ is a parameter. The simplest way to accomplish this goal is to uniformly sample $x$ over the available space, and let $dx = range(x)/n_{events}$. However, if $f(x)$ spans several orders of magnitude over the range of $x$, it is more advantageous to preferentially sample the values of x that lead to larger $f(x)$. If we sample $x$ with probability distribution function of $\rho(x)$, then we perform

$I = \int f(x) \rho(x) dx^{'} \quad(1a)$

$dx^{'} = \dfrac{range(x)}{n_{events} \rho(x)} \quad (1b)$

Performing the integral in this manner will allow us to save time, provided we pick a good value for $\rho(x)$.

For randomly selecting energies, we will use $\rho(E_{\nu}) \sim E_{\nu}^{-2}$. This is harder than the power spectrum at low energies and softer than the power spectrum at high energies, but it is still better than uniform sampling. To keep the probability density normalized ($\int_{E_{min}}^{E_{max}} \rho(E) dE = 1$), we'll use

$\rho(E_{\nu}) = \kappa E_{\nu}^{-2} \quad;\quad \kappa = \dfrac{E_{min}E_{max}}{E_{max}-E_{min}}\quad(3)$

To get energies that fit this distribution, we will use $X \sim U[0,1]$ and find the energy $E_{\nu}$ such that

$\int_{E_{min}}^{E_{\nu}} \kappa E^{-2} dE = X \quad (4a)$

$E_{\nu} = \dfrac{\kappa E_{\min}}{\kappa - E_{\min} X} \quad(4b)$

We would like to now sample volume elements, but do it in such a way that is favorable for the Monte Carlo. We want our events to be within $5\lambda$ of the detector (any further than this, and the probability for detection drops drastically. We also need our events to occur within the Earth. For any two overlapping spheres of radii $r_1$ and $r_2$ with their centers separated by distance $d$ ($d < r_1 + r_2$), the volume of the overlap is

$V_{overlap} = \dfrac{\pi}{12d}(r_1 + r_2 - d)^2 \big(\, d^2 + 2d(r_1+r_2) - 3(r_1-r_2)^2 \big )\, \quad(6a)$

And in our case where the center of one sphere is on the surface of another ($d = r_1$)

$V_{overlap} = \dfrac{\pi}{12r_1}(r_2)^2 \big(\, 8r_1r_2 - 3r_1^2 \big )\, \quad(6b)$

Therefore for our case of $r_1 = R_{\oplus}$ and $r_2 = 5 \lambda$

$V_{interaction} = \dfrac{\pi}{12R_{\oplus}}(5\lambda)^2(40R_{\oplus}\lambda - 75 \lambda^2) \quad for \quad 5 \lambda < 2 R_{\oplus} \quad (6c)$

$V_{interaction} = \dfrac{4\pi R_{\oplus}^3}{3} \quad for \quad 5 \lambda \geq 2R_{\oplus}$

We will weight our results by $dV^{'} = dV ( V_{interaction}/V_{\oplus})$

To select volume elements, we'll use spherical coordinates $r^{'}, \theta^{'}, \phi^{'}$ relative to the detector, with the true position vector given by $\overrightarrow{x} = \big(\, r^{'} \sin\theta^{'} \cos\phi^{'}, r^{'} \sin\theta^{'} \sin\phi^{'}, R_{\oplus} + r^{'} \cos\theta^{'} \big )\,$. We sample these values from $\phi^{'} \sim U[0,2\pi]$, $\cos\theta^{'} \sim U[-1,0]$ (we only want angles that point toward the Earth), $r^{'} = 5 \lambda \sqrt[3]{X}$ where $X \sim U[0,1]$. (If $5\lambda > 2 R_{\oplus}$ then $r^{'} = 2 R_{\oplus} \sqrt[3]{X}$)

We then require $|\overrightarrow{x}| \leq R_{\oplus}$. If this is not the case, we sample again, and repeat this process until all of the events have been assigned a position

We now wish to calculate $\dfrac{d\sigma}{d\cos{\beta}}$, where $\beta$ is the scattering angle, which will be used in the probability to scatter. We have a function for $\dfrac{d\sigma}{dt}$ where $t = -q^2$ and $q$ is the transferred momentum. We will work in the infinite mass limit, so the energy transfered to the nucleus is zero, and $E_N = E_{\nu}$. Therefore, we have

$t = -(\overrightarrow{p_{\nu}} - \overrightarrow{p_{N}})^2 = E_{\nu}^2+p_N^2-2E_{\nu}p_N \cos\beta \quad (9)$

Where we know

$p_N = \sqrt{E_N^2-m_N^2} = \sqrt{E_{\nu}^2-m_N^2} \quad(10)$

$t = 2E_{\nu}^2 - m_N^2 - 2 E_{\nu}\sqrt{E_{\nu}^2-m_N^2} \cos\beta \quad(11a)$

$dt = -2 E_{\nu} \sqrt{E_{\nu}^2 - m_N^2} d(\cos\beta) \quad (11b)$

We can now sub this into our differential cross section

$\dfrac{d\sigma}{dt} = \dfrac{Z^2 d^2 \alpha}{t E_{\nu}^2} (4E^2_{\nu} -m^2_{N}+\dfrac{m^4_N}{t}) \quad (12a)$

$\dfrac{d\sigma}{d(-2E_{\nu}\sqrt{E_{\nu}^2 - m_N^2}\cos\beta)} = \dfrac{Z^2 d^2 \alpha}{t E_{\nu}^2} (4E^2_{\nu} -m^2_{N}+\dfrac{m^4_N}{t}) \quad (12b)$

$\dfrac{d\sigma}{d(\cos\beta)} = \dfrac{-2\sqrt{E_{\nu}^2-m_N^2}*Z^2 d^2 \alpha}{E_{\nu}t} (4E^2_{\nu} -m^2_{N}+\dfrac{m^4_N}{t}) \quad (12c)$

$\dfrac{d\sigma}{d(\cos\beta)} = \dfrac{-2\sqrt{E_{\nu}^2-m_N^2}*Z^2 d^2 \alpha}{E_{\nu} (2E_{\nu}^2 - m_N^2 - 2 E_{\nu}\sqrt{E_{\nu}^2-m_N^2} \cos\beta)} (4E^2_{\nu} -m^2_{N}+\dfrac{m^4_N}{2E_{\nu}^2 - m_N^2 - 2 E_{\nu}\sqrt{E_{\nu}^2-m_N^2} \cos\beta}) \quad (12d)$

This returns a negative answer, so we flip the sign

The flux of incoming neutrinos is isotropic, so the true distibution of scattering angles $\beta$ follows $\cos\beta \sim U[-1,1]$. Since the value of $\dfrac{d\sigma}{d\cos\beta}$ become orders of magnitude larger for a small range of $\cos\beta$ it makes sense to only sample where the cross section is large. We can see that the differential cross section roughly follows $\dfrac{1}{1-\cos\beta}$. For right now, we'll define $A = 1-\cos\beta$ and let

$\rho(A) = \dfrac{\epsilon}{A} \quad ; \quad \epsilon = \dfrac{1}{\ln(A_{max}/A_{min})} \quad (13)$

Our true values for $A_{min}$ and $A_{max}$ are 0 and 2, respectively. However, plugging 0 into the expression causes trouble, so we will define a UV cutoff for our $A_{min}$ (currently at $10^{-5}$). To randomly sample this distribution, let $X \sim U[0,1]$ and

$\int_{A_{min}}^{A} \dfrac{\epsilon}{A^{'}} dA^{'} = X \quad (14a)$

$A = (A_{min})^{1-X} (A_{max})^{X} \quad (14b)$

$\cos\beta = 1 - (A_{min})^{1-X} (A_{max})^{X} \quad (14c)$

Now, we will $\psi \sim U[0,2\pi]$. We will use $\psi$ and $\beta$ to define $\overrightarrow{v_{in}}$ which is the path by which the neutrino entered Earth. This is will become important when considering matter effects for neutrino oscillations.

$\hat{v_{in}} = \hat{v_{out}}\cos\beta + \hat{v_{\perp1}}\sin\beta\cos\psi + \hat{v_{\perp2}}\sin\beta\sin\psi \quad (15)$

Where $\hat{v_{out}}$,$\hat{v_{\perp1}}$, and $\hat{v_{\perp2}}$ are all mutually orthogonal. Initial definition of the two new vectors is arbitrary, so we shall let

$\dfrac{\overrightarrow{x} \times \overrightarrow{v_{out}}}{|\overrightarrow{x} \times \overrightarrow{v_{out}}|} = \hat{v_{\perp1}} \quad (16a)$

$\hat{v_{out}} \times \hat{v_{\perp1}} = \hat{v_{\perp2}} \quad(16b)$

Where $\overrightarrow{v_{out}}$ is the vector that points from the volume element to the detector. It is given by $\overrightarrow{v_{out}} = (-x, -y, R_{\oplus} - z)$ where $x$, $y$, and $z$ are the Cartesian coordinates of the volume element

We also need $n(\overrightarrow{x})$, which is the number density of nuclei at our volume element. We claim density is purely a function of radius r out from the center of the Earth, and that it linearly decreases as the radius increases. Setting the density of the surface to 3 $g/cm^3$ and the density at the core to 13 $g/cm^3$ gives us a mass similar to that of the Earth

def Earth_Density(r, R_Earth):
    core_density = 13 #g/cm^3
    surface_density = 3 #g/cm^3
    
    dens = core_density + (surface_density - core_density) * (r/R_Earth)
    return(dens)

The contribution to our total rate of decays is

$dR = A_{\perp}\dfrac{d\sigma}{d\cos\beta} n(\overrightarrow{x})\Phi(E_{\nu}) \dfrac{P_{dec}(\overrightarrow{v_{out}},\lambda)}{4\pi |\overrightarrow{v_{out}}|^2} dV dE_{\nu} d\cos\beta \quad (18a)$

$dV = \dfrac{\dfrac{4}{3}\pi R_{\oplus}^3}{\sqrt[3]{n_{events}}} \quad ; \quad dE_{\nu} = \dfrac{E_{max}-E_{min}}{\sqrt[3]{n_{events}}} \quad ; \quad d\cos\beta = \dfrac{2}{\sqrt[3]{n_{events}}} \quad (18b) $

For uniformly sampled values. Since we did preferential sampling, we will have to replace $dV$, $dE_{\nu}$, and $d\cos\beta$ with $dV^{'}$, $dE_{\nu}^{'}$, $d\cos\beta^{'}$. For the variable where we just restrict the parameter spaces ($dV^{'}$) it is relatively simple

$dV^{'} = \dfrac{V_{interaction}}{\sqrt[3]{n_{events}}} \quad(19) $

For the parameters we sampled from a power law, we want

$dE_{\nu}^{'} = \dfrac{E_{max}-E_{min}}{\sqrt[3]{n_{events}}} \bigg (\, \dfrac{E_{\nu}}{E_{char}} \bigg )\, ^2 \quad d\cos\beta^{'} = \dfrac{2}{\sqrt[3]{n_{events}}} \dfrac{A}{A_{char}} \quad (20)$

Where $A = 1-\cos\beta$. To find this characteristic energy and angle, we want to find where $\rho(E_{\nu}) = \dfrac{1}{E_{max}-E_{min}}$, $\rho(A) = 1/2$ where $\rho(E_{\nu})$ is from Eq 3 and the RHS of the equations are the uniform probability distributions

$E_{char} = \sqrt{E_{min}E_{max}} \quad A_{char} = \dfrac{A_{max}-A_{min}}{ln(A_{max}/A_{min})} \quad (21)$

We now have all the information needed to calculate the rate

Saving 1D Density Profiles for re-weighting¶

If we want to consider flavor dependence, we will need to calculate the density profile that the neutrino passed through to reach the volume element and apply flavor oscillations to that. For now, we are considering a single element and a density that solely depends on radius

If our interaction occurs at the volume element $\overrightarrow{x}$ and the neutrino has an incoming vector of $\overrightarrow{v_{in}}$ with a unit vector given by Eq 1, the we know

$|\overrightarrow{x} - \overrightarrow{v_{in}}|^2 = R_{\oplus}^2 \quad (22a)$

$|\overrightarrow{x}|^2 - 2 \overrightarrow{x} \cdot \hat{v_{in}} |\overrightarrow{v_{in}}| + |\overrightarrow{v_{in}}|^2 = R_{\oplus}^2 \quad (22b)$

Using the quadratic formula, we know that

$|\overrightarrow{v_{in}}| = \overrightarrow{x} \cdot \hat{v_{in}} \pm \sqrt{ (\overrightarrow{x} \cdot \hat{v_{in}})^2 -4 (|\overrightarrow{x}|^2 - R_{\oplus}^2)} \quad (23)$

Since $R_{\oplus} \geq |\overrightarrow{x}|$, the value in the square root will always be greater than the first term. Since magnitude is strickly positive, we will take the plus sign. Using the unit vector and the magnitude, we can now calculate $\overrightarrow{v_{in}}$. From this, we know that the path traveled is

$\overrightarrow{r_{\nu}(u)} = \overrightarrow{x} + (u-1)\overrightarrow{v_{in}} \quad;\quad u \in [0,1] \quad (24)$

And the distance traversed by the neutrino at any value of u is $d = u |\overrightarrow{v_{in}}|$. The distance from the neutrino to the center of the Earth can be found by

$|\overrightarrow{r_{\nu}(u)}|^2 = |\overrightarrow{x}|^2 + (u-1)^2 |\overrightarrow{v_{in}}|^2 + 2 (u-1) \overrightarrow{x} \cdot \overrightarrow{v_{in}} \quad (25)$

We can then plug this into our function for the density

def One_D_Density_Profile(x_vect_vals, v_in_hats, R_Earth):
    '''
    This function takes in vector for the volume element
    and the unit vector for the incoming neutrino and returns
    the corresponding density profiles
    
    The inputs for the vectors must be 2D arrays with
    the first index refering the event number and the second 
    index refering to the cartesian dimension
    
    This returns the distances and densities as 2D arrays
    with the first index refering to event number and the 
    second index refering to the path
    '''
    
    #Find the dot product of the two vectors
    x_dot_vhat = x_vect_vals[:,0]*v_in_hats[:,0] + x_vect_vals[:,1]*v_in_hats[:,1] + x_vect_vals[:,2]*v_in_hats[:,2]
    x_mag = np.sqrt(x_vect_vals[:,0]*x_vect_vals[:,0] + x_vect_vals[:,1]*x_vect_vals[:,1] + x_vect_vals[:,2]*x_vect_vals[:,2])
    #print('x*v', x_dot_vhat)
    
    #Find the magnitude of vin
    vin_mag = x_dot_vhat + np.sqrt(x_dot_vhat**2 -(x_mag**2 - R_Earth**2))
    #print('vin mag/R_E', vin_mag/R_Earth)
    
    #Determine the distances and densities
    u_vals = np.linspace(0,1,10000)
    distances = np.zeros((len(vin_mag),len(u_vals)))
    densities = np.zeros((len(vin_mag),len(u_vals)))
    u_index = 0
    for u in u_vals:
        distances[:,u_index] = vin_mag*u
        r_2 = x_mag**2 + (u-1)**2 *vin_mag**2 + 2 *(u-1) * x_dot_vhat* vin_mag  #cm^2
        rad = np.sqrt(r_2) #cm
        #print(r/R_Earth)
        densities[:,u_index] = Earth_Density(rad, R_Earth)
        u_index += 1
    return(distances,densities)

Determining the zenith angle of the neutrino's entry¶

This code uses an isotropic flux of neutrinos. However, the true flux of neutrinos depends upon the zenith angle of the neutrino's entry (the angle is defined so $\phi_{zenith} = 0$ corresponds to a neutrino entering normal to Earth's surface

We will call the position of entry $\overrightarrow{r_{entry}}$ and define it as

$\overrightarrow{r_{entry}} = \overrightarrow{x} - \overrightarrow{v_{in}} \quad (26)$

We can find the zenith angle by

$\cos\phi_{zenith} = \hat{r_{entry}} \cdot \hat{v_{in}} = \dfrac{(\overrightarrow{x}-\overrightarrow{v_{in}}) \cdot (-\hat{v_{in}})}{|\overrightarrow{r_{entry}}|} = \dfrac{|\overrightarrow{v_{in}}| - \overrightarrow{x}\cdot \hat{v_{in}}}{R_{\oplus}} \quad (27)$

def Zenith_Angles(x_vect_vals, v_in_hats, R_Earth):
    '''
    This function takes in the 2D arrays for position and 
    incoming neutrino direction unit vectors.  The first index
    corresponds to the event number, while the second index
    correpsonds to the cartesian coordiante.
    The function also reads in a float for the radius of the Earth.
    This returns a 1D arraw of zenith angles'''
    #Find the dot product of the two vectors
    x_dot_vhat = x_vect_vals[:,0]*v_in_hats[:,0] + x_vect_vals[:,1]*v_in_hats[:,1] + x_vect_vals[:,2]*v_in_hats[:,2]
    x_mag = np.sqrt(x_vect_vals[:,0]*x_vect_vals[:,0] + x_vect_vals[:,1]*x_vect_vals[:,1] + x_vect_vals[:,2]*x_vect_vals[:,2])
    #print('x*v', x_dot_vhat)
    
    #Find the magnitude of vin
    vin_mag = x_dot_vhat + np.sqrt(x_dot_vhat**2 -(x_mag**2 - R_Earth**2))
    
    cos_zeniths = (vin_mag - x_dot_vhat)/R_Earth
    zeniths = np.arccos(cos_zeniths)
    return(zeniths)
